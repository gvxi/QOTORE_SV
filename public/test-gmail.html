üîß Gmail API Setup Guide

Follow these steps to configure Gmail API for order notifications:

  1. Create Google Cloud Project
    Go to Google Cloud Console and create a new project
  2. Enable Gmail API
    In APIs & Services ‚Üí Library, search for "Gmail API" and enable it
  3. Create OAuth2 Credentials
    In APIs & Services ‚Üí Credentials, create OAuth client ID (Web application type)
  4. Configure OAuth Consent Screen
    Add app information and your Gmail as a test user
  5. Generate Refresh Token
    Use OAuth Playground with gmail.send scope
  6. Configure Environment Variables
    Add these to your Cloudflare Workers dashboard:
    ADMIN_EMAIL = your-admin@email.com
    GMAIL_USER = your-gmail@gmail.com
    GOOGLE_CLIENT_ID = your-client-id.apps.googleusercontent.com
    GOOGLE_CLIENT_SECRET = GOCSPX-your-client-secret
    GOOGLE_REFRESH_TOKEN = 1//your-refresh-token
    SITE_URL = https://yourdomain.com
  7. Test Configuration
    Use the "Send Test Email" button above to verify everything works
‚ö†Ô∏è Important: The OAuth refresh token needs gmail.send scope. Make sure to select this scope in OAuth Playground when generating your token.
